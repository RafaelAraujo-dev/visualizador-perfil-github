# 🔍 Visualizador de Perfil do GitHub

Um projeto simples e funcional que utiliza a API do GitHub para buscar e exibir informações de perfis de usuários, como avatar, bio e repositórios.

![GitHub Preview](https://img.shields.io/badge/GitHub-API-blue?style=for-the-badge&logo=github)
![Status](https://img.shields.io/badge/Status-Concluído-brightgreen?style=for-the-badge)

---

## 🚀 Funcionalidades

- **Busca por Usuário:** Digite o nome de usuário do GitHub para ver as informações.
- **Atalho no Teclado:** Realize a busca pressionando a tecla **Enter**.
- **Exibição de Dados:** Mostra a foto de perfil, bio, nome e repositórios públicos.
- **Feedback de Carregamento:** Exibe uma mensagem de "Carregando..." enquanto os dados são buscados.
- **Tratamento de Erros:** Alerta o usuário caso o perfil não seja encontrado ou haja algum erro na busca.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **HTML5:** Para a estrutura da página.
- **CSS3:** Para a estilização, incluindo animações e responsividade.
- **JavaScript (ES6+):** Para a lógica de busca e consumo da API do GitHub (Fetch API).
- **Devicons:** Para os ícones de tecnologia.

## ⚙️ Como Executar o Projeto

1.  Clone este repositório para o seu computador:
    ```bash
    git clone https://github.com/seu-usuario/visualizador-perfil-github.git
    ```
2.  Navegue até a pasta do projeto:
    ```bash
    cd visualizador-perfil-github
    ```
3.  Abra o arquivo `index.html` em seu navegador (ou use a extensão "Live Server" no VS Code).
